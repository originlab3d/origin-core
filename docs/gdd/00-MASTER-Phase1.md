# ORIGIN — MASTER GDD PHASE 1
**Kirou Chat Tamagotchi | Oct 2026 Launch**

---

## VISION EXÉCUTIVE

**ORIGIN Phase 1** = Kirou Chat, un compagnon tamagotchi intelligent avec IA, monétisé via vente d'œufs et items in-game.

**Objectif** : 200,000 L$/mois d'ici Décembre 2026

---

## BUSINESS MODEL

### Revenue Streams

| Source | Prix | Cible | Monthly L$ |
|--------|------|-------|-----------|
| Kirou Starter (Œuf) | 750-1500 L$ | 100-180 joueurs | 112K-270K L$ |
| Skins à débloquer | Flux ou L$ | - | +20-30K L$ |
| Animation Packs | Flux ou L$ | - | +10-15K L$ |
| **TOTAL** | - | - | **200K L$** |

### Monétisation

- **Œuf Kirou** : 750-1500 L$ (achat SL → compte bancaire)
- **Flux** : Monnaie in-game
  - Gagnée via : interactions pet, quêtes, events
  - Dépensée pour : packs skins, animations, soins spécialisés
- **Premium Items** : Certains skins/gestures premium payants en L$

### Cost Structure

- **Railway hosting** : ~$15/mois
- **Gemini API** : Paiement à l'usage (dépend load)
- **Estimé** : $50-100/mois en infra

**Margin** : ~90%+ après costs

---

## CIBLE UTILISATEURS

- **Joueurs SL** : Amateurs de pets/compagnons
- **Âge** : 18-45 ans
- **Intérêt** : Roleplay social, collection, personnalisation
